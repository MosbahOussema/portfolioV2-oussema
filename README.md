# Oussama Mosbah Portfolio

Personal portfolio built with React and Vite to present my frontend experience, selected projects, services, and contact channels.

## Tech Stack

- React 18
- Vite
- CSS modules by component
- EmailJS
- React Toastify
- Material UI icons

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env.local` and fill the EmailJS/contact values:

```bash
VITE_EMAILJS_SERVICE_ID=
VITE_EMAILJS_TEMPLATE_ID=
VITE_EMAILJS_PUBLIC_KEY=
VITE_CONTACT_EMAIL=
VITE_LINKEDIN_URL=
VITE_GITHUB_URL=
VITE_WHATSAPP_PHONE=
```

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
- `npm run preview` previews the production build locally.
- `npm run lint` checks the code with ESLint.
